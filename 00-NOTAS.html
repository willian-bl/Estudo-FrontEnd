1. HYPERTEXTO E A INTERNET Hypertexto não é uma leitura linear. É uma leitura interativa, onde o leitor escolhe qual caminho ele quer seguir e sobre quais assuntos quer ler. Uma página aponta para outra e tudo está conectado, ou seja, linkado. A internet funciona no sistema cliente/servidor. Toda vez que acessamos uma página na internet, o cliente faz uma requisição (pedido) ao servidor para que ele entregue essa página para nós. O servidor responde e devolve um HTML, a página que pedimos. E o nosso navegador interpreta essa página e nos mostra o resultado. URL -> Localizador de recursos HTTP -> Hyper Text Transfer Protocol (Protocolo de Transferência de Hypertexto): Protocolo padrão para a transferência e comunicação entre as máquinas na internet. HTML -> Hyper Text Markup Language (Linguagem de Marcação de Hypertexto): Linguagem utilizada para construir as páginas da WEB, posicionando e marcando onde cada conteúdo deve ficar e como deve ser exibido pelo navegador (se é um título, um parágrafo, uma imagem etc). *Atenção: HTML não é uma linguagem de programação, é uma linguagem de marcação. Ninguém PROGRAMA em HTML. Por volta dos anos 2000, as tabelas eram a base dos sites. Menus e páginas eram todos feitos a partir de tabelas, pois era o recurso que tinham a disposição. Além disso, os navegadores interpretavam o HTML de maneira diferente um do outro. W3C -> Consórcio de várias empresas que definem os padrões da internet. *Usos: HTML -> conteúdo CSS -> estilo/formatação JavaScript -> interatividade 2. HTML - INFORMAÇÕES BÁSICAS São arquivos de texto com marcações. Tem extensão .html Para escrever precisamos de um editor de código O arquivo é lido (interpretado) através dos navegadores (browsers). Ex: Mozilla Firefox, Google Chrome etc Uma página WEB é um arquivo HTML. Já um site é a junção de vários arquivos HTML index.html é o nome padrão do arquivo da página principal do site. Quando temos um diretório que contém um arquivo index.html o browser reconhece e carrega esse arquivo quando não especificamos o caminho de nenhum outro 3. Ex: Visite nosso site! A Tag de link PRECISA, obrigatoriamente, de no mínimo um atributo, que é para qual site o link leva. Cada tag tem seus próprios atributos que podem ou não ser obrigatórios e para saber quais são, podemos utilizar uma colinha de algum site ou, normalmente, um editor de código mostra os atributos que podemos usar. É claro que, com o tempo e a prática, acabamos decorando os atributos mais comuns. 4. ESTURUTURA BÁSICA Todo arquivo html apresenta a seguinte estrutura: Declaração do tipo de arquivo.
